왜 Eduino AI 인가요?

why eduino ai

AI를 글과 그림으로만 배우면 막연합니다. Eduino AI는 학습자가 직접 데이터를 만들고, 모델을 학습시키고, 실시간으로 반응을 확인하게 합니다. 추상적인 개념을 손으로 만지는 경험으로 바꾸는 것이 핵심입니다.

쉬운 이해

K-12 눈높이의 학습 정보·알고리즘 정보 카드로 "이게 왜 이렇게 동작하지?"를 단계마다 풀어 설명합니다.

체험 기반 빠른 이해

카메라·소리·데이터로 즉시 반응. 보고 만지며 학습해 한 번의 수업으로 원리를 체감합니다.

교실 상호작용

몸으로 참여하는 활동, 실시간 게이지·도넛 피드백, 단계 진행 표시로 교사-학생 활동을 설계합니다.

3단계 학습 설계

learning tiers

모든 모듈은 같은 흐름으로 이어집니다. 관찰로 시작해, 내 손으로 AI를 만들고, 실생활 문제에 응용합니다.

초급 · 체험

관찰하고 원리를 이해

사전학습 모델을 그대로 사용하며 시각화로 관찰합니다.

  • 비추기만 하면 즉시 인식
  • 결과를 게이지·오버레이로 확인
  • "AI가 어떻게 보는가" 체감
중급 · 제작

내 데이터로 AI 만들기

직접 데이터를 모아 전이학습하고 파라미터를 조정합니다.

  • 클래스 만들고 샘플 수집
  • 학습 곡선·혼동행렬 확인
  • "내 AI"의 정확도 끌어올리기
심화 · 응용

평가하고 실생활에 응용

일반화·평가·연동으로 깊은 개념과 실전 활용까지.

  • 과적합·일반화 실험
  • 파라미터로 성능 비교
  • 실생활 미니 프로젝트

이용 방법

how it works

준비물은 인터넷이 연결된 PC/태블릿과 웹캠(또는 마이크)뿐입니다.

1

모듈 선택

학습 목표에 맞는 모듈을 고릅니다. 설치·로그인 없이 링크 클릭만으로 바로 시작합니다.

2

체험 → 제작

사전학습 모델로 먼저 관찰하고, 내 데이터를 모아 직접 AI를 학습시킵니다.

3

확인 · 응용

실시간 피드백으로 결과를 확인하고, 파라미터를 바꿔가며 실생활 문제에 응용합니다.

자주 묻는 질문

faq
설치하거나 가입해야 하나요?
아니요. 모든 모듈은 브라우저에서 바로 동작합니다. 별도 설치·회원가입 없이 링크만으로 사용할 수 있습니다.
학생 데이터(사진·음성)는 서버로 전송되나요?
전송되지 않습니다. AI 학습과 추론은 모두 사용자의 기기 안(On-Device)에서 처리됩니다. 카메라·마이크 데이터는 외부로 나가지 않습니다.
카메라가 켜지지 않아요.
브라우저 보안 정책상 카메라는 localhost 또는 HTTPS 환경에서만 켜집니다. 배포된 https 주소로 접속하거나, 로컬에서는 localhost 주소로 열어주세요. 접속 후 카메라 권한을 허용해야 합니다.
어떤 기기가 필요한가요?
인터넷이 연결된 PC 또는 태블릿과 웹캠/마이크면 충분합니다. 인터페이스는 태블릿 화면에도 맞게 반응형으로 동작합니다.
수업에서 어떻게 활용하나요?
초급(체험)으로 흥미를 끌고, 중급(제작)에서 학생이 직접 데이터를 모아 AI를 만들며, 심화(응용)에서 실생활 문제로 확장합니다. 가위바위보·실시간 데이터 수집 같은 상호작용 활동을 함께 제공합니다.
