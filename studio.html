AI를 보고, 직접 만든다

카메라로 AI가 세상을 어떻게 보는지 관찰하고, 내 손으로 나만의 AI를 학습시켜 보세요. 두 가지 트랙으로 자유롭게 오갈 수 있어요.

🔍
Track 1 · 관찰

기본기능 체험하기

학습 없이 클릭 한 번으로 AI를 작동시켜, 컴퓨터가 화면 속 세상을 어떻게 인식하는지 직접 관찰해요. 수업 도입에 딱 좋아요.

  • 객체 탐지 COCO-SSD
  • 얼굴 검출 BlazeFace
  • 뼈대 검출 MoveNet
  • 색상 추적
  • 영상 효과
준비물 없이 바로 체험 체험 시작 →
🛠
Track 2 · 창작

직접 제작하기

내가 직접 데이터를 모아 AI를 학습시키고, 결과를 평가해요. 클래스 만들기 → 데이터 수집 → 학습 → 추론까지, 나만의 AI를 완성합니다.

  • 이미지 분류 MobileNet
  • 음성 분류 Speech Commands
  • 동작 분류 MoveNet
내 AI 만들기 제작 시작 →

직접 제작하기는 이렇게 진행돼요

STEP 1

클래스 만들기

구분하고 싶은 종류를 정해요. 예: 캔·페트·종이

STEP 2

데이터 수집

각 종류마다 사진을 여러 장 모아요. 많을수록 똑똑해져요.

STEP 3

모델 학습

모은 데이터로 나만의 AI를 학습시켜요.

STEP 4

추론·평가

새 사진으로 정확도를 확인해요. 제출·평가 곧 추가
